# Sistema de Horas Extras · BAMOCOL S.A.

Sistema de registro y clasificación de horas extras según la 
normativa colombiana vigente (Ley 2466 de 2025 · CST).

## ¿Qué hace este sistema?

Permite a un supervisor registrar las horas extras de los empleados.
El sistema clasifica automáticamente cada hora como diurna, nocturna,
dominical o festiva, acumula los totales por semana y mes, y genera
un reporte listo para el liquidador de nómina.

## Normativa aplicada

- Ley 2466 de 2025 — jornada nocturna desde las 7:00 p.m.
- CST art. 162 — límite de 2 h diarias y 12 h semanales
- Recargo dominical: 90% en 2026, 100% desde julio 2027
